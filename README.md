<h2>Cara Install</h2>
Tool ini menggunakan python3 

<pre>git clone https://github.com/Bandhitawkunthi/TawkunDoS</pre>

Install request:
```
$ pip3 install pysocks
$ pip3 install bs4
```
Install Proxy:
```
$ cd TawkunDoS
$ python3 TawkunProxy.py
```
Install Socks:
```
$ cd TawkunDoS
$ python3 TawkunSOCKS.py
```
Runn:
```
$ cd TawkunDoS
$ python3 Tawkun.py
```
